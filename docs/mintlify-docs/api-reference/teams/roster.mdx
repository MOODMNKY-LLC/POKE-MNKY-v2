# Team Roster

Get a team's roster (active picks) and budget information for a specific season.

## Request

<ParamField path="teamId" type="string" required>
  Team UUID
</ParamField>

<ParamField query="seasonId" type="string" required>
  Season UUID
</ParamField>

## Response

<ResponseField name="team_id" type="string">
  Team UUID
</ResponseField>

<ResponseField name="season_id" type="string">
  Season UUID
</ResponseField>

<ResponseField name="team_name" type="string">
  Team name
</ResponseField>

<ResponseField name="roster" type="array">
  Array of active draft picks, each containing:
  - `draft_pick_id`: Draft pick UUID
  - `pokemon_id`: Pokémon UUID
  - `pokemon_name`: Pokémon name
  - `pokemon`: Full Pokémon object with types, draft points, etc.
  - `points_snapshot`: Points spent on this pick
  - `acquisition`: How acquired (draft, free_agency, trade)
  - `status`: Pick status (active, dropped, traded)
  - `draft_round`: Round number
  - `pick_number`: Pick number
</ResponseField>

<ResponseField name="budget" type="object">
  Budget information:
  - `points_used`: Total points spent
  - `budget_total`: Total budget (from season config)
  - `budget_remaining`: Points remaining
  - `slots_used`: Roster slots used
  - `slots_total`: Maximum roster size
  - `slots_remaining`: Roster slots remaining
</ResponseField>

## Example Request

```bash
curl -X GET "https://poke-mnky.moodmnky.com/api/teams/team-uuid-here/roster?seasonId=123e4567-e89b-12d3-a456-426614174000"
```

## Error Responses

- `400` - Invalid UUID format or missing seasonId
- `404` - Team or season not found
- `500` - Server error
