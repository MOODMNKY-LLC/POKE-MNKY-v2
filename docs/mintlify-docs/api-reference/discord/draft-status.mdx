# Discord Draft Status

Get the current draft status for a guild/season.

## Request

<ParamField query="guild_id" type="string" required>
  Discord guild/server ID
</ParamField>

<ParamField query="season_id" type="string">
  Season UUID (optional, uses guild default if not provided)
</ParamField>

## Response

<ResponseField name="ok" type="boolean">
  Success status
</ResponseField>

<ResponseField name="season" type="object">
  Season information:
  - `id`: Season UUID
  - `name`: Season name
  - `status`: Draft status (drafting, completed, not_started)
</ResponseField>

<ResponseField name="current_pick" type="object">
  Current pick information:
  - `team_id`: Team UUID whose turn it is
  - `team_name`: Team name
  - `coach_name`: Coach name
  - `pick_number`: Current pick number
  - `round`: Current round number
</ResponseField>

<ResponseField name="draft_stats" type="object">
  Draft statistics:
  - `total_picks`: Total picks made
  - `picks_remaining`: Picks remaining
  - `teams_participating`: Number of teams
</ResponseField>

## Example Request

```bash
curl -X GET "https://poke-mnky.moodmnky.com/api/discord/draft/status?guild_id=123456789012345678" \
  -H "Authorization: Bearer YOUR_DISCORD_BOT_API_KEY"
```
