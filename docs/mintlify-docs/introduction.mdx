# Welcome to POKE MNKY

POKE MNKY is a comprehensive Pokémon Draft League Management Platform built with modern web technologies.

## What is POKE MNKY?

POKE MNKY is a full-featured platform for managing Pokémon draft leagues, including:

- **Draft Management** - Complete draft system with point budgets and roster management
- **Team Building** - Team builder with validation and Showdown integration
- **Battle Management** - Match scheduling, results tracking, and standings
- **Discord Integration** - Bot commands for draft picks, team management, and notifications
- **Notion Sync** - Bidirectional sync with Notion databases for league data
- **AI Features** - OpenAI-powered assistants for draft strategy and battle analysis

## Key Features

### Draft System
- Point-based draft with customizable budgets
- Real-time draft room with live updates
- Draft pool management with role tags and type effectiveness
- Transaction audit trail

### Team Management
- Roster validation against league rules
- Budget tracking and calculations
- Free agency transactions
- Team export/import

### Battle Integration
- Pokémon Showdown integration
- Battle replay storage and analysis
- Match result tracking
- Standings and statistics

### Discord Bot
- Slash commands for draft picks
- Team status and roster queries
- Coverage analysis
- Guild-specific configuration

### Notion Sync
- Bidirectional data synchronization
- Deterministic mapping system
- Incremental sync support
- Job tracking and monitoring

## Technology Stack

- **Frontend**: Next.js 16 (App Router) with React 19.2
- **Backend**: Supabase (PostgreSQL) with Row Level Security
- **AI**: OpenAI GPT-4.1, GPT-5.2, GPT-5-mini
- **Integrations**: Discord.js, Pokémon Showdown, Notion API
- **Deployment**: Vercel (app) + Homelab (services)
