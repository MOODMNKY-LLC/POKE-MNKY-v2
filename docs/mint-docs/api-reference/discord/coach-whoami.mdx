# Discord Coach Whoami

Get coach information by Discord user ID.

## Request

<ParamField query="discord_user_id" type="string" required>
  Discord user ID
</ParamField>

<ParamField query="season_id" type="string">
  Season UUID (optional, uses current season if not provided)
</ParamField>

## Response

<ResponseField name="ok" type="boolean">
  Success status
</ResponseField>

<ResponseField name="coach" type="object">
  Coach information:
  - `id`: Coach UUID
  - `discord_user_id`: Discord user ID
  - `name`: Coach name
  - `team_id`: Team UUID (if assigned to a team)
  - `team_name`: Team name (if assigned)
</ResponseField>

<ResponseField name="team" type="object">
  Team information (if coach has a team):
  - `id`: Team UUID
  - `team_name`: Team name
  - `season_id`: Season UUID
  - `roster_count`: Number of Pok√©mon on roster
  - `points_used`: Points spent
  - `points_remaining`: Points remaining
</ResponseField>

## Example Request

```bash
curl -X GET "https://poke-mnky.moodmnky.com/api/discord/coach/whoami?discord_user_id=123456789012345678" \
  -H "Authorization: Bearer YOUR_DISCORD_BOT_API_KEY"
```

    "id": "team-uuid",

## Error Responses

- `404` - Coach not found
- `401` - Unauthorized (invalid/missing bot key)
